Package gppLibrary.functionals.composites

Package gppLibrary.functionals.composites Description

The gpp.functionals.composites package provides easily reused combinations of pipelines and groups.
A pipeline of groups provides a Task parallel architecture in which each stage is itself a group of Data parallel worker processes.
A group of pipelines provides a Data parallel architecture in which each worker is itself a Task parallel implementation
