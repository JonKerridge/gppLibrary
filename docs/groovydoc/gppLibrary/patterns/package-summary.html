Package gppLibrary.patterns

Package gppLibrary.patterns Description

Package gpp.patterns defines processes that can used as is simply by making appropriate reference to the data objects used in the application. The programmer is required to make no other changes. As such the package patterns contains some commonly occurring parallel structures. On occasions programmers will need to create their own parallel architectures as a response to the requirements of the application.

The package gpp_demos.MCpi} shows how the Farm pattern is used with different combinations of data object.
